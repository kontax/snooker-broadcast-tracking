The goal of tagging these images is to get training data for the neural network. We need to detect whether the the image being shown is the full table from the main view (where the camera is facing straight down the table), and the full table is shown without any missing edges. If so, then press the "Yes" button, otherwise choose "No".

If the same tag is selected 3 times for an image it will disappear, however if a tag is different from something that was selected previously, then all the tags for the image are reset.
